## Быстрая установка
* Открыть программу Intellij IDEA  
* Нажать на кнопку «Get from Version Control»
* Вставить URL ссылку: `https://github.com/mistervenik/ChsvGroup.git`

### Требования
* Java 11
* Java FX 14
* Intellij IDEA 2020.2.3
